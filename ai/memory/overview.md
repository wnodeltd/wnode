# Wnode — Overview

## What is Wnode?

Wnode is the Planetary Compute Mesh — a decentralised network that turns idle consumer devices (phones, laptops, EVs, smart TVs) into shared compute infrastructure. Participants ("Nodlrs") earn real daily income via Stripe payouts.

## Key Principles

- **No crypto, no wallets** — payouts in real currency.
- **Community-owned** — governed by a DAO with 1 Soul = 1 Vote.
- **Lightweight** — nodes run via a WASM-based operator app.

## Current Status

- MVP is live and open for beta testers.
- AI Seed is in training mode, learning from project documentation.
