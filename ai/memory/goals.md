# Wnode — Goals

## Immediate Goals (AI Seed Phase)

1. **Stand up the AI Seed** — a lightweight, CPU-only model that can read project docs and generate insights.
2. **Integrate with Command Dashboard** — display AI status, indexed file count, and latest insight.
3. **Establish the memory pipeline** — build tooling to convert structured data into .md files the model can consume.

## Medium-Term Goals

1. **Expand memory coverage** — index operational data, mesh telemetry, and governance records.
2. **Improve inference quality** — upgrade to a larger model once Mac Studio rack hardware is available.
3. **Automate insight generation** — periodic refresh cycles that surface actionable intelligence.

## Long-Term Vision

- The AI becomes a core operational layer of the mesh.
- Real-time anomaly detection, capacity planning, and governance advisory.
- Community-contributed memory files expand the knowledge base.
