# Wnode AI Models

Only the dev team should install models manually on the server.

The model file must be placed manually in this directory.
File name must be: `tiny-local-model.bin`

No model files should ever be committed to Git.

## ONNX Model Requirements
- Model must be ONNX format (.onnx)
- File name MUST be: tiny-local-model.onnx
- File is installed manually on the server by the dev team
- No model files are committed to Git
