# CMD Documentation Screenshots Checklist

This file tracks the required screenshots for the Command Portal documentation.

## Required Assets

- [ ] `dashboard-overview.png`
  - Source: `/` (Dashboard)
  - Focus: Vitals cards, Operational Metrics, and Fleet Map.
- [ ] `crm-slideout.png`
  - Source: `/crm`
  - Focus: Selected user row with the detail slide-out open.
- [ ] `ledger-statements.png`
  - Source: `/ledger`
  - Focus: Transaction list and monthly statement grouping.
- [ ] `nodes-overview.png`
  - Source: `/nodls` (or `/inventory`)
  - Focus: Node list with status indicators and health metrics.
- [ ] `cmd-navigation.png`
  - Source: Any page
  - Focus: Sidebar highlighting and Sovereign panel layout.

## Instructions
- Capture in 2x resolution (Retina).
- Use Dark Mode theme.
- Ensure no sensitive PII is visible (use test data).
- Save to `docs/cmd/screenshots/` directory.
