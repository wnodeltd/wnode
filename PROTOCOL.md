# Wnode Protocol Constitution

## Sales Source Amendment
Wnode is a decentralized compute mesh where node operators (Nodlrs) act as the sales force. The 'Sales Source' logic ensures that if a Nodlr brings a client to the mesh, they are rewarded perpetually, even if their own hardware isn't used for that specific job.

### Revenue Distribution (Sales Source Active)
For jobs where a 'Sales Source' is identified, the revenue split is as follows:

| Recipient | Allocation | Role |
| :--- | :--- | :--- |
| **Compute (Nodlr)** | **70%** | The node operator providing the hardware |
| **Sales Source** | **10%** | The Perpetual Growth Agent who introduced the client |
| **L1 Affiliate** | **3%** | Immediate direct referral (Level 1) |
| **L2 Affiliate** | **7%** | Secondary referral (Level 2) |
| **Founder** | **3%** | Sovereign Founder share |
| **Protocol (Wnode)** | **7%** | Platform maintenance and protocol development |

Total: 100%
